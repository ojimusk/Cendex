# Cendex

> Modern Perpetual DEX built on Hyperliquid.

Cendex is a next-generation decentralized perpetual exchange focused on speed, simplicity, and professional trading experience.

Our vision is to build an exchange that combines the performance of centralized exchanges with the transparency and security of decentralized finance.

---

# Vision

Create the most accessible perpetual trading platform for everyone.

We believe professional trading should be:

- Fast
- Secure
- Transparent
- Easy to use

---

# Why Cendex?

Most decentralized exchanges are difficult for beginners.

Cendex aims to solve this by providing:

- Clean UI
- Lightning-fast execution
- Deep liquidity
- Professional trading tools
- Self-custody wallet
